# Employee Management System
A console-based Employee Management System built using Python.
The application allows users to manage employee records, perform analytics, and store data in a JSON file.

## Features
- Add Employee
- View All Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee
- Sort Employees by Salary
- View Total Employees
- Find Highest Paid Employee
- Find Lowest Paid Employee
- Calculate Average Salary
- Filter Employees by Department
- Filter Employees Above Salary
- Group Employees by Department
